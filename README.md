## Deployment

### Recommended: Vercel
This app is built with Next.js and is optimized for Vercel.
1. Go to [Vercel.com](https://vercel.com)
2. Sign up/Login with GitHub.
3. Click **"Add New..."** -> **"Project"**.
4. Import `premium-life-os`.
5. Click **Deploy**.

### GitHub Pages
To deploy to GitHub Pages, you must configure `output: 'export'` in `next.config.ts`, but some dynamic features (like Image Optimization) may not work. Vercel is highly recommended.
